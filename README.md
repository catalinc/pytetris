# pytetris
Tetris clone written with Python and Pygame.

_WIP_

### Requirements
* Python >= 2.7
* Pygame >= 1.9.2

### Instructions
Run `python game.py` to start the game.
Controls:
* ←,↓,→ - move piece
* z - rotate piece CW
* x - rotate piece CCW
* Space - drop piece
