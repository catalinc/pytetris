# pytetris
Tetris clone written with Python and Pygame.

![demo](https://github.com/catalinc/pytetris/raw/master/demo.png)

### Requirements
* Python >= 2.7
* Pygame >= 1.9.2

### Instructions
Run `python game.py` to start the game.

#### Controls
* ←,↓,→ - move piece
* z - rotate piece CW
* x - rotate piece CCW
* space - drop piece
* p - pause game
* r - resume (new) game
